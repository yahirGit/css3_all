CSS bordes

Bordes en CSS

Border Shorthand

la propiedad border tiene la caracteristica de ser un ShortHand que permite definir el borde para cada lado del modelo de caja con un tamaño y estilo diferente. 

            
                border-width: 10px 25px 5px 15px;
                border-style: solid dotted inset double;
                border-color: indigo blue darkolivegreen darksalmon;

Como en este ejemplo anteior cada borde del modelo de caja tiene un estilo diferente.

tambien se puede ser mucho mas especifico en definir el estilo de un borde pudiendo hacer lo siguiente.

            
                border-top-width: 10px;
                border-top-style: double;
                border-top-color: black;

Como podemos ver en este ejemplo usamos una forma mas especifica para definir el borde superior de la caja. Esto tambien es posible en cada lado del modelo de caja.

CSS border radius

Border radius example

La propiedad border radius tiene la caracteristica de ser un ShortHand lo que permite ser mas especifico para definir el border de cada lado del modelo de caja. 

            
                border-top-left-radius: 20px;
                border-top-right-radius: 20px;
                border-bottom-right-radius: 20px;
                border-bottom-left-radius: 20px;

Para redondear completamente una caja el tamaño de la misma debe ser proporcional, es decir el mismo alto y ancho debemos definir su tamaño, como veremos a continuación. 

            
                border-radius: 50%;

CSS Box Shadow

Box shadow

La propiedad box shadow permite crear sombras en los diferentes elementos.

La propiedad box-shadow cuenta con los siguientes valores. 

            
                box-shadow: h-offset v-offset blur spread color;

Donde explicamos acontinuación cada una de ellas.

  • offset-x: Define el desplazamiento de la sombra en el eje x, pude tener valores negativos.
  • offset-y: Define el desplazamiento de la sombra en el eje y, puede tener valores negativos.
  • blur radius: Define la difuminacion de la sombra entre mayor sea el valor mas grande, suave y ligera sera la sombra.
  • spread radius: Define que la sombra se expanda y crezca, permite valores negativos.
  • color: define el color de la sombra.
