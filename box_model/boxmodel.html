hablaremos del modelo de caja. lo fundamental

El modelo de caja se entiene como el comportamiento de las etiquetas html dentro de un documento html, toda etiqueta html tiene por defento una propiedad display que establece como se debe comportar esa etiqueta display: block display:inline suelen ser los valores por defecto para todas las etiquetas html.

Pero esto establece que cada etiqueta se comportar como una caja en si, es decir cada contenido que este dentro de una etiqueta html esta establecido en un recuadro encerrado para ese contenido que define su tamaño ya sea de manera explicita es decir nosotros le asignamos un alto y un ancho, o implicita donde el navegador decide ese tamaño.

hablaremos del modelo de caja. Margin

El Margin Establece un espacio entre las etiquetas contiguas este espaciado puede ser en valores positivos o negativos.

La propiedad margin tiene una caracteristica es una propiead Shorthand lo cual permite definir el espaciado en todos los sentido en dos dimenciones es decir puedes aplicar margen top,right,bottom,left = superior,derecho,inferior,izquierdo en ese orden, existen otras propiedades css que tienen esa caracteristica. margin

Asi se aplican un margen usando la caracteristica de Shorthand. 

                h1{
                    margin: top right bottom left;
                    margin: 2px 10px 10px 15px;    
                }
            
                h1{
                    margin: top right bottom left;
                    margin-top: 50px;
                    margin-right: 10px;
                    margin-bottom: 50px;
                    margin-left: 50px;
                }
